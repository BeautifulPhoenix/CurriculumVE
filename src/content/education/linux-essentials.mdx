## Key Learnings

- Linux System Architecture: Understanding of Linux system components and boot process.

- Command Line: Proficiency in shell commands and basic scripting for system administration.

- File System Management: Navigation and management of Linux file systems and permissions.

- Package Management: Installation and maintenance of software using package management tools.

- System Security: Implementation of basic security practices and user management.

- Process Management: Understanding and control of Linux processes and services.
